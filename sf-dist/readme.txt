

Requirements:
	JRE 1.6
	Ant (1.8.1 see ./bin)

Intellij Requirements:
  * Intellij IDEA (Community Edition) 
	http://www.jetbrains.com/idea/download/

  * JsChilicat Plugin 
	Can be installed via Settings or can be downloaded here:
	http://plugins.intellij.net/plugin/?id=5006


Start test execution with ANT
  * Add ANT to the classpath (./bin/ant). See ANT documentation for details: http://ant.apache.org/manual/install.html
  * Just execute ant in the commandline: "ant"
  * or "ant google-chrome" to start test execution in google chrome browser.
  * or "ant firefox" to start test execution in firefox browser.


Try out intellij test execution
  * Download and install latest version of intellij (Community Edition).
  * Start intellij.
  * Intall jschilicat plugin: 
	1. Open settings dialog: File/Settings
	2. Search in settings dialog for "plugins"
	3. Click in "Available" tab.
	4. Search for jschilicat
	5. Install plugin.
	6. Restart intellij
  * Open test project: intellij-test-project
  * Run "JsChilicat Example". See dropbox in the the tool bar.
